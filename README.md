## Características de la Aplicación
1. Gestión de Tableros, Listas y Tarjetas.
2. Tipos de tarjetas (Tareas y Checklists).
3. Etiquetas de colores.
4. Historial de acciones y bloqueo temporal de tableros.
5. Acceso mediante URL única por email.

**Características Opcionales Implementadas:**
- Reglas a nivel de lista (Max N items y validación de listas previas).
- Filtrado de tarjetas por etiquetas.
- Creación de plantillas YAML.
- Compactación automática de tableros.

## Tecnologías
- Java 21, Spring Boot, Spring Data JPA, H2 Database.
- Interfaz gráfica con JavaFX y FxWeaver.
- Maven como sistema de construcción.
- Arquitectura Hexagonal y Domain-Driven Design (DDD).
